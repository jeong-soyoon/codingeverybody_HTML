HTML

Hypertext Markup Language (HTML)

  • strong 태그: 강조하고자 하는 부분 감쌈
  • u 태그: underline의 u로 밑줄치고자 하는 부분 감쌈
  • h1 태그 : heading의 h로 제목 태그, 숫자가 낮을수록 크고 높을수록 작다. (h1~h6)
  • br 태그
    • 줄바꿈 태그
    • 무언가를 설명하지 않기에 감싸지 않고 태그를 닫지 않는다.
    • 여러 번 사용해서 단락 간 간격을 맘대로 조절가능
  • p 태그
    • paragraph의 p로 단락을 감쌈
    • 단락의 경계를 분명히 하지만 단락 간 간격이 일정함->css로 해결 : p 태그에 style ="margin-top: 45px"를 추가, p 태그 위쪽에 45px만큼의 여백(margin)이 생김
  • img 태그
    • image의 img 태그와 source의 src라는 속성(태그이름만으로 정보가 부족) -> img src=
    • 인터넷에 있는 이미지를 삽입하려면 속성 값으로 이미지 주소를 넣어주고,
    • 내 컴퓨터에 있는 이미지를 삽입하려면 저장 후 프로젝트폴더에 위치, 속성 값으로 이미지 파일 이름 넣어줌
    • 이미지 크기 조정 : img 태그안에 width = "100%"를 넣어 % 앞의 값을 조정하며 크기를 조정가능
  • li 태그 : list의 li로 목록 한 줄마다 감싸서 목록 표현, ol태그 또는 ul태그를 필요로 함
  • ul 태그 : unordered list의 ul로 목록 전체를 감쌈, 항목 번호 x(검정동그라미),li태그를 필요로 함
  • ol 태그 : ordered list의 ol로 목록 전체를 감쌈, 항목 번호 o, li태그를 필요로 함
  • title 태그: 웹페이지 제목을 감싸서 지정
  • meta charset = "utf-8" 태그
    • 문자열부분에 웹페이지가 저장된 문자표현방식 써주면 됨
    • 웹페이지가 저장된 문자 표현 방식과 웹브라우저가 웹페이지를 해석하는 방식이 일치해서 영어가 아닌 문자 깨지지 않게
  • head 태그 : 본문을 설명하는 태그 ex)title 태그 , meta charset =
  • body 태그 : 본문 태그
  • !doctype html 태그 : 이 웹페이지가 HTML로 만들어졌다는 것을 표현하기 위해 문서 시작에 추가
  • html 태그 : body와 head를 감싸는 하나의 태그
  • a 태그
    • a href = "" target = "_blank" title =""
    • anchor의 a로 링크 걸고 싶은 문자를 감쌈
    • href는 hypertext reference의 약자로 ""안에 페이지 주소를 넣어줌
    • target = "_blank"는 링크를 클릭했을 떄 새창에서 페이지가 열리게 함
    • title는 링크가 어떤 내용을 담고 있는지를 ""안에 넣어줌
